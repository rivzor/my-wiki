    Активация Windows/Office

    Внимание

    Руководство по активации Windows приведено только для ознакомления и не является призывом к действию

    Пошаговая инструкция по активации Windows и Office с использованием Microsoft Activation Scripts (MAS). Скрипты позволяют активировать систему и офисные продукты разными методами: HWID, Ohook, TSforge и Online KMS.

    Запуск скрипта

    1. Откройте PowerShell (Пуск → PowerShell).

    2. Вставьте команду:

      • Win 11/10/8 
        irm https://get.activated.win | iex
      • Win 7 
        iex ((New-Object Net.WebClient).DownloadString('https://get.activated.win'))
      Если команда блокируется провайдером/DNS

      Win 11/10: 

      iex (curl.exe -s --doh-url https://1.1.1.1/dns-query https://get.activated.win | Out-String)

      Win 7 

      iex ((New-Object Net.WebClient).DownloadString('https://get.activated.win'))

    3. Появится меню активации. Выберите зелёные пункты для активации Windows или Office.

    4. Готово!
    Если скрипт все равно не запускается
    1. Скачать файл: MAS_AIO.cmd
    2. Запустить MAS_AIO.cmd

    Методы активации

    Метод Продукты Срок активации Интернет
    HWID Windows 10–11 Постоянно Да
    Ohook Office Постоянно Нет
    TSforge Windows / ESU / Office Постоянно Да (на новых сборках)
    Online KMS Windows / Office 180 дней (с автообновлением) Да
