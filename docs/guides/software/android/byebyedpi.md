# ByeByeDPI
![ByeByeDPI](https://github.com/romanvht/ByeDPIAndroid/raw/master/.github/images/app.svg){ align=left  width="70"}
ByeByeDPI — инструмент для обхода блокировок на Android. Оно использует SOCKS5-прокси для перенаправления трафика локально, не шифруя данные и не скрывая IP.

### Возможности

- Обход DPI и цензуры
- Локальный VPN-режим без удаленного сервера
- Перенаправление трафика через SOCKS5-прокси
- Настройка параметров для разных стратегий
- Поддержка Android TV и роутеров (с настройкой)
