# PeaZip

![PeaZip](https://upload.wikimedia.org/wikipedia/commons/thumb/f/fe/Peazip_ico.svg/1200px-Peazip_ico.svg.png){ align=left width="70" }
PeaZip — кросс-платформенный архиватор с функциями шифрования, сжатия и управления файлами. Доступен в portable и installable версиях.


### Возможности

- Поддержка 200+ форматов архивов
- Шифрование (AES, Twofish, Serpent)
- Проверка целостности и восстановление
- Интеграция с проводником
- Portable-режим без установки
