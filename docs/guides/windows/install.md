# Установка Windows с помощью Rufus

Этот гайд - пошаговая инструкция по чистой установке Windows (10 или 11) с нуля. Мы будем использовать Rufus для создания загрузочной флешки.

!!! info "Обзор"
    * **Что делаем?** Чистая установка Windows.
    * **Инструмент:** Rufus.
    * **Требования:** ПК с UEFI (для Win 11 также TPM 2.0 и Secure Boot).
    * **Время:** 30-60 минут.

---

## Шаг 1: Подготовка

Сперва нужно собрать все необходимое.

-   **ISO-образ Windows:** Нужен оригинальный образ. Для Windows 11 убедитесь, что ваш ПК подходит по системным требованиям (процессор Intel 8+ gen / AMD Ryzen 2000+).
-   **Rufus:** Скачайте последнюю версию с официального сайта. Portable-версия удобнее, так как не требует установки.
-   **USB-накопитель:** Флешка на 8 ГБ или больше. **Все данные на ней будут удалены.** Лучше взять USB 3.0, так будет быстрее.

!!! warning "Бэкап!"
    Перед установкой **обязательно** скопируйте все важные данные с ПК. Процесс установки форматирует диск.

## Шаг 2: Создание загрузочной USB

1.  Вставьте флешку в ПК.
2.  Запустите Rufus (лучше от имени администратора).
3.  Настройте параметры в программе:
    -   **Устройство (Device):** Выберите вашу флешку. Убедитесь, что это она, чтобы не стереть другой диск.
    -   **Метод загрузки (Boot selection):** Нажмите "ВЫБРАТЬ" (SELECT) и укажите путь к вашему ISO-образу Windows.
    -   **Схема раздела (Partition scheme):** **GPT**. Это стандарт для UEFI. (MBR нужен только для очень старых ПК с Legacy BIOS).
    -   **Целевая система (Target system):** **UEFI (non CSM)**.
    -   **Файловая система:** **NTFS**.
4.  Нажмите "СТАРТ". Rufus может предложить скачать доп. файлы (Syslinux) - соглашайтесь.
5.  Процесс займет 5-10 минут.

??? "Совет для Windows 11 (Обход TPM)"
    Если ваш ПК не соответствует требованиям (нет TPM или Secure Boot), Rufus 4.x сам это определит и предложит опцию "Remove requirements for..." (Убрать требования). Оставьте эту галочку включенной.

---

## Шаг 3: Настройка BIOS/UEFI

1.  Перезагрузите ПК. Как только он начнет включаться, жмите клавишу входа в BIOS. Обычно это `Del`, `F2`, `F10` или `Esc`.
2.  Найдите раздел "Boot" (Загрузка).
3.  Установите вашу USB-флешку на **первое место** в порядке загрузки (Boot Order / Boot Priority).
4.  (Для Windows 11) Убедитесь, что **Secure Boot** включен (если Rufus не обходит это).
5.  Сохраните настройки и выйдите (обычно `F10` -> `Enter`).

---

## Шаг 4: Установка Windows

1.  ПК перезагрузится и автоматически начнет загрузку с флешки.
2.  Появится установщик Windows.
3.  Выберите язык, время и раскладку. Нажмите "Далее".
4.  "Установить" (Install now).
5.  Введите ключ (или нажмите "У меня нет ключа", чтобы активировать позже).
6.  Выберите редакцию (Home / Pro).
7.  Примите лицензию.
8.  **Тип установки:** Выберите **"Выборочная" (Custom)**. Это и есть чистая установка.
9.  **Выбор диска:** Вы увидите список разделов.
    -   Если диск чистый, просто выберите "Незанятое пространство" и жмите "Далее".
    -   Если на диске уже стояла система, **удалите** все старые разделы (System, Recovery, MSR, Primary), пока не останется один пункт "Незанятое пространство". Выберите его и жмите "Далее".
10. Начнется копирование файлов.

---

## Шаг 5: После установки

Windows будет копировать файлы и несколько раз перезагрузится.

1.  После перезагрузок начнется настройка "OOBE" (выбор региона, подключение к сети, создание аккаунта).
2.  Как только попадете на рабочий стол, первым делом проверьте драйверы.
3.  Зайдите в "Центр обновления Windows" (Windows Update) и запустите поиск. Он скачает большинство драйверов.
4.  Драйвер для видеокарты (NVIDIA/AMD/Intel) лучше скачать с официального сайта.
5.  Активируйте Windows в Настройках.

!!! success "Готово"
    Windows установлена. Если что-то пошло не так (ошибка копирования файлов) - попробуйте другую флешку или проверьте скачанный ISO-образ.
