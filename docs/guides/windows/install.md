# Установка Windows с помощью Rufus

!!! info "Обзор"
    Эта инструкция охватывает правильную установку Windows (10/11) с нуля с использованием Rufus. Требования: ПК с UEFI (для Windows 11 — TPM 2.0, Secure Boot). Время: 30–60 мин. Rufus — бесплатный инструмент для создания загрузочной USB.

## Шаг 1: Подготовка материалов
- **ISO-образ Windows:** Скачайте с официального сайта Microsoft (media creation tool для 10/11). Размер: ~5–6 ГБ. Для Windows 11 проверьте системные требования (процессор Intel 8+ поколения или AMD Ryzen 2000+).
- **Rufus:** Скачайте последнюю версию (4.x на 2025 год) с rufus.ie. Portable-версия не требует установки.
- **USB-накопитель:** Минимум 8 ГБ, пустой (данные сотрутся). Рекомендую USB 3.0+ для скорости.

!!! warning "Предупреждение"
    Сделайте бэкап данных на ПК. Установка сотрёт диск.

## Шаг 2: Создание загрузочной USB
1. Вставьте USB в ПК.
2. Запустите Rufus (как администратор).
3. В интерфейсе:
   - **Device:** Выберите ваш USB (проверьте имя, чтобы не стереть другой диск).
   - **Boot selection:** Нажмите "SELECT" и выберите ISO Windows.
   - **Image option:** "Standard Windows installation".
   - **Partition scheme:** GPT (для UEFI) или MBR (для legacy BIOS).
   - **Target system:** UEFI (non CSM) для современных ПК.
   - **File system:** NTFS.
   - **Cluster size:** Default.
4. Нажмите "START". Процесс: 5–10 мин. Rufus автоматически настроит всё для установки.

??? "Совет для Windows 11"
    Если ПК не соответствует требованиям (нет TPM), используйте Rufus с опцией "Extended Windows 11 installation" (обходит проверки).

## Шаг 3: Настройка BIOS/UEFI для загрузки
1. Перезагрузите ПК и войдите в BIOS (F2/Del/Esc — зависит от материнки).
2. В Boot-меню: Установите USB как первый в порядке загрузки.
3. Включите Secure Boot (для Windows 11).
4. Сохраните (F10 + Enter) и выйдите.

## Шаг 4: Установка Windows
1. Загрузитесь с USB (ПК перезагрузится автоматически).
2. В установщике:
   - Выберите язык, время, клавиатуру → "Next".
   - "Install now".
   - Введите ключ продукта (или "I don't have" для активации позже).
   - Выберите редакцию (Home/Pro).
   - Примите лицензию.
   - **Тип установки:** Custom (для чистой установки).
   - Выберите диск: Удалите разделы (если нужно), создайте новые → "Next".
3. Установка: 10–20 мин (ПК перезагрузится несколько раз).
4. После: Настройте аккаунт, подключите интернет, обновите систему.

## Шаг 5: После установки
- Установите драйверы (с сайта производителя ПК).
- Активируйте Windows в Настройках → Update & Security → Activation.
- Обновите через Windows Update.

!!! success "Готово"
    Windows установлена. Если ошибки — проверьте ISO на целостность (SHA-хэш с сайта Microsoft).
