# Базовая оптимизация Windows

!!! info "Обзор"
    Это "чек-лист" действий, которые нужно выполнить сразу после чистой установки Windows. Цель: убрать телеметрию, отключить ненужные службы, настроить систему на максимальную производительность.

---

## Шаг 1: Обновления и Драйверы

Первый шаг после установки — приведение системы в актуальное состояние.

### 1.1. Центр обновления Windows

1.  Открыть `Параметры` -> `Центр обновления Windows` (Windows Update).
2.  Нажать "Проверить наличие обновлений".
3.  Установить **все** доступные обновления, включая "Необязательные" (Optional Updates) — там часто лежат драйверы чипсета и звука.
4.  Перезагружаться столько раз, сколько потребуется.

### 1.2. Драйверы Видеокарты

Windows Update ставит базовый драйвер. Для игр и производительности нужен полный пакет.

* **NVIDIA:** Скачать GeForce Experience или драйвер с [официального сайта](https://www.nvidia.com/Download/index.aspx).
* **AMD:** Скачать Adrenalin Software с [официального сайта](https://www.amd.com/en/support).
* **Intel Arc:** Скачать с [сайта Intel](https://www.intel.com/content/www/us/en/download-center/home.html).

### 1.3. Проверка Диспетчера устройств

1.  Нажать `Win + X` и выбрать `Диспетчер устройств` (Device Manager).
2.  Проверить, что в списке нет устройств с желтым восклицательным знаком.
3.  Если есть — найти драйвер на сайте производителя ноутбука или материнской платы.

## Шаг 2: Приватность и Телеметрия

Отключаем "слежку" Microsoft.

1.  Открыть `Параметры` -> `Приватность и защита` (Privacy & Security).
2.  **Общие (General):** Отключить **все 4** переключателя (Рекламный идентификатор, отслеживание запуска и т.д.).
3.  **Диагностика и отзывы (Diagnostics & feedback):**
    * Отключить "Отправлять необязательные данные".
    * Отключить "Улучшение рукописного ввода".
    * Отключить "Персонализированные возможности".
    * Установить "Частота отправки отзывов" -> "Никогда".
4.  **Журнал действий (Activity history):** Снять галку "Сохранять журнал действий на этом устройстве".
5.  **Расположение (Location):** Отключить, если не используется ноутбук и карты.
6.  **Камера, Микрофон:** Проверить, каким приложениям дан доступ. Отключить у ненужных.

---

## Шаг 3: Очистка от "Мусора" (Bloatware)

Удаляем предустановленный хлам.

1.  Открыть `Параметры` -> `Приложения` -> `Установленные приложения` (Apps -> Installed apps).
2.  Удалить все, что не нужно.
    * `Xbox` (если не используется).
    * `Teams` (если не нужен).
    * `OneDrive` (если не используется).
    * `Feedback Hub`, `Get Help`, `People`, `Cortana`.
    * Сторонний антивирус (если он был, например, McAfee) — встроенный Defender работает отлично.

??? question "Удаление 'неудаляемых' приложений (PowerShell)"
    Некоторые приложения (как Cortana) не удаляются через `Параметры`. Для этого используется PowerShell (от им. Администратора).

    ```powershell
    # Пример удаления Cortana
    Get-AppxPackage *Microsoft.549981C3F5F10* | Remove-AppxPackage
    
    # Пример удаления Xbox (всех компонентов)
    Get-AppxPackage *xbox* | Remove-AppxPackage
    ```

## Шаг 4: Настройка Проводника (Explorer)

Делаем Проводник удобным.

1.  Открыть Проводник.
2.  Нажать на `...` (или меню `Вид`) -> `Параметры` (Options).
3.  Вкладка **"Вид" (View)**:
    * [ВКЛ] "Всегда отображать значки, а не эскизы" (Always show icons, never thumbnails) — *по желанию, ускоряет открытие папок с фото/видео*.
    * [ВКЛ] "Показывать расширения для зарегистрированных типов файлов" (Show file name extensions). **(Обязательно)**
    * [ВКЛ] "Показывать скрытые файлы, папки и диски" (Show hidden files...).
    * [ОТКЛ] "Скрывать защищенные системные файлы" (Hide protected operating system files) — *снять галку (по желанию, для продвинутых)*.
4.  Нажать "Применить к папкам" (Apply to Folders).

## Шаг 5: Оптимизация Производительности

### 5.1. Визуальные эффекты

Отключение анимации и прозрачности ускоряет отклик интерфейса.

1.  Открыть `Параметры` -> `Персонализация` -> `Цвета`.
2.  Отключить **"Эффекты прозрачности" (Transparency effects)**.
3.  Открыть `Параметры` -> `Специальные возможности` (Accessibility) -> `Визуальные эффекты`.
4.  Отключить **"Эффекты анимации" (Animation effects)**.

??? info "Тонкая настройка (Панель управления)"
    Для более детальной настройки:
    
    1.  Нажать `Win + R`, ввести `systempropertiesperformance` и нажать Enter.
    2.  Выбрать "Обеспечить наилучшее быстродействие" (Adjust for best performance).
    3.  Оставить только 3 галки:
        * "Вывод эскизов вместо значков" (Show thumbnails instead of icons).
        * "Использование стилей отображения для окон и кнопок" (Use visual styles...).
        * "Сглаживание неровностей экранных шрифтов" (Smooth edges of screen fonts).

### 5.2. План электропитания

1.  Нажать `Win + R`, ввести `powercfg.cpl` и нажать Enter.
2.  Выбрать "Максимальная производительность" (High performance).
3.  Если ее нет, выбрать "Сбалансированная" (Balanced).

### 5.3. Автозагрузка

Чем меньше программ стартует с Windows, тем быстрее загрузка.

1.  Нажать `Ctrl + Shift + Esc` (Диспетчер задач).
2.  Перейти на вкладку `Автозагрузка` (Startup) или `Приложения автозагрузки`.
3.  Отключить все, что не является критически важным (Steam, Discord, Spotify, Update-чекеры и т.д.). Оставить только драйверы (Realtek, NVIDIA) и антивирус.

### 5.4. Защита системы (System Restore)

По умолчанию включена и "ест" место на диске.

1.  Нажать `Win + R`, ввести `sysdm.cpl` и нажать Enter.
2.  Перейти на вкладку "Защита системы" (System Protection).
3.  Выбрать диск (C:) и нажать "Настроить" (Configure).
4.  Выбрать "Отключить защиту системы" (Disable) и удалить все точки.

!!! warning "Внимание"
    Отключение Защиты системы не позволит "откатить" Windows в случае сбоя драйвера или обновления. Это действие для тех, кто предпочитает бэкапы (Acronis/Macrium) или быструю переустановку.

## Шаг 6: Контроль учетных записей (UAC)

UAC — это надоедливое окно "Разрешить этому приложению...".

1.  Нажать `Win`, ввести `UAC`, открыть "Изменение параметров контроля учетных записей".
2.  Опустить ползунок на один или два пункта вниз (до "Не уведомлять при внесении изменений в параметры Windows").
3.  Установка в самый низ ("Никогда не уведомлять") **не рекомендуется** из соображений безопасности.

!!! success "Готово"
    Система настроена. После выполнения этих шагов рекомендуется **перезагрузить ПК**.
